# Ownership and Use Notice

This repository is public for deployment and collaboration purposes, but the project is **not**
open source.

## Summary
- Source code, UI design, branding, and content are proprietary.
- Copying or re-hosting the site/code without permission is not allowed.
- Public availability on GitHub/GitHub Pages does **not** grant reuse rights.

## Security / Privacy Policy for This Repo
- Treat **all committed files** as public.
- Treat **all files in `site/`** as public web output.
- Do not commit secrets, tokens, credentials, internal operational data, or private notes.
- Runtime state should be stored outside public site outputs and should not be tracked in Git.

## Reporting
If you discover a security issue or accidental exposure, open a private report to the repository owner
instead of posting secrets publicly in an issue.
